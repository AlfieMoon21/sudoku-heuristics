# Sudoku Heuristics Comparison

## Setup

1. Download sudoku dataset from Kaggle
2. Place CSV file in `data/sudoku.csv`
3. Install dependencies: `python3 -m pip install pandas numpy matplotlib`
4. Run: `python3 sudoku_solver.py`

## Dataset

Download from: [Kaggle Sudoku Dataset](https://www.kaggle.com/datasets/)
https://www.kaggle.com/datasets/bryanpark/sudoku?resource=download
(Search for "sudoku" and get CSV with `quizzes` and `solutions` columns)
